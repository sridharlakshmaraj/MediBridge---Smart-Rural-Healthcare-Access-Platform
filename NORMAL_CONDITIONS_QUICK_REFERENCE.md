### Category: Common Infections (5 Conditions)

**1. Cold 🤧**
```
Symptoms: Runny nose, sneezing, sore throat, cough, fatigue
Duration: 5-7 days
Medications: Antihistamines, decongestants, pain relievers, honey
Query Examples:
  - "What are symptoms of a cold?"
  - "What medications treat a cold?"
  - "Cold home remedies?"
  - "How long does a cold last?"
```

**2. Fever 🌡️**
```
Symptoms: High temperature, chills, sweating, weakness, headache
Duration: 2-3 days typically
Medications: Acetaminophen, ibuprofen, cool compress, hydration
Query Examples:
  - "What is a fever?"
  - "How to reduce fever?"
  - "Fever medications?"
  - "When fever requires doctor?"
```

**3. Sore Throat 🗣️**
```

Medications: Pain relievers, throat lozenges, honey tea, sprays
Query Examples:

  - "Medications for sore throat?"
  - "Natural sore throat relief?"
```

**4. Headache 🤕**
```
Symptoms: Head pain, pressure, stiffness, sensitivity to light

Medications: Ibuprofen, acetaminophen, aspirin, cold/warm compress
Query Examples:
  - "What causes headaches?"
  - "How to treat headache?"
  - "Headache medications?"
  - "Migraine vs tension headache?"
```

**5. Cough 🤧**
```
Symptoms: Coughing reflex, throat irritation, chest discomfort
Duration: 2-3 weeks possible
Medications: Cough suppressant, expectorant, honey, throat lozenges
Query Examples:
  - "What is a cough?"
  - "Treatments for cough?"
  - "Dry vs wet cough?"
  - "When to see doctor for cough?"
```
